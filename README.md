## Sobre

Faça upload de uma planilha exportada do Azure Portal (`.csv`, `.xlsx`, `.xls`) e veja instantaneamente quais recursos podem ser movidos.

**Funcionalidades:**
- 📊 Dashboard com resumo (movíveis, parciais, não movíveis)
- 🔍 Pesquisa e filtros por nome, tipo ou status
- 📤 Exportação dos resultados em CSV
- 🌙 Tema claro e escuro
- 🌐 8 idiomas (EN, PT-BR, 中文, ES, FR, العربية, RU, 日本語)

## Como Usar

1. Exporte seus recursos do **Azure Portal** em `.csv` ou `.xlsx`
2. Acesse [cloudmoveanalyzer.com](https://cloudmoveanalyzer.com) ou abra `index.html` localmente
3. Arraste o arquivo para a área de upload
4. Pronto — veja o resultado na tabela
