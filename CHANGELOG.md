## [1.6.0] - 2026-06-18

### Added — Security hardening
- **Content-Security-Policy** (currently `Report-Only`; promoted to enforce after a
  validation window) covering scripts, styles, images, fonts, connections, frames,
  workers, and `frame-ancestors`. Locks down `object-src`, `base-uri`, `form-action`.
- **Cross-Origin-Embedder-Policy: credentialless** added to existing COOP/CORP set
  for a fully-isolated browsing context.
- **Upload magic-byte validation** in `processFile()`: rejects spoofed extensions
  (XLSX zip signature `50 4B 03 04`, XLS compound binary `D0 CF 11 E0`, NUL-byte
  sniff for binary content masquerading as CSV).
- **Upload extension allow-list** (`csv`, `tsv`, `xls`, `xlsx`) before parsing.
- **`safeHttpUrl()`** wrapper for any `<a href>` rendered into the table — blocks
  `javascript:` / `data:` / `vbscript:` defense-in-depth.
- **Self-hosted flag SVGs (`assets/flags/*.png`)** and **ko-fi badge
  (`assets/kofi-badge.png`)** — removes runtime dependency on
  `flagcdn.com` and `storage.ko-fi.com`, tightens the CSP `img-src` allow-list,
  and eliminates a third-party CDN supply-chain risk.
- **`SECURITY.md`** hardening summary section listing every mitigation in place.

### Added — Performance
- **Per-language i18n bundles** (`js/i18n/<lang>.js`) generated by
  `scripts/split-i18n.js` from the canonical `data/i18n-source.js`. Initial
  payload drops from ~130 KB (all 8 langs) to ~28 KB (en + one language).
- **`scripts/split-i18n.js`** invoked automatically by `scripts/build.js`.
- **`<link rel="preload" as="script">`** for the 4 critical scripts — parallelizes
  download with HTML parse.
- **`scripts/optimize-images.ps1`** — PNG re-encoder and resizer (System.Drawing,
  no external dependencies). Originals preserved under `assets/_originals/`
  (gitignored).
- **`assets/favicon-32.png`** (1.3 KB) — dedicated 32×32 variant; previously the
  128×128 master was served for both DPI tiers.

### Changed — Performance
- **`assets/logo.png`**: resized 1984×2012 → 512×512 and re-encoded.
  **925 KB → 88 KB (−90%)**.
- **`renderTable()`** rewritten: single `innerHTML` assignment from a pre-built
  string array (was per-row `<tr>` element + DocumentFragment). Noticeably faster
  on tables with 1000+ rows.
- **AdSense (`adsbygoogle.js`)** moved out of `<head>` and now loaded inside a
  `requestIdleCallback` after `load`, so it never competes with the app for
  initial bandwidth.
- **i18n core (`js/i18n.js`)** rewritten as a slim loader (~2 KB); language data
  lives in `data/i18n-source.js` (canonical) → `js/i18n/<lang>.js` (generated).
- `scripts/build.js` runs `split-i18n.js` as part of every build.

### Changed — Security
- **`MAX_FILE_SIZE_BYTES`** lowered from 100 MB to 10 MB (Azure Portal CSV
  exports are typically <2 MB; cap blocks memory-exhaustion attempts).
- Per-language pages built by `scripts/build.js` now inject the matching
  `js/i18n/<lang>.js` script and matching preload hint.

### Changed — UX
- "Buy me a coffee" CTA softened across all 8 languages — heading changed from
  imperative ("☕ Buy me a coffee!") to invitational ("☕ Enjoyed the tool?"),
  and the supporting sentence simplified.

### Removed
- Runtime dependency on `https://flagcdn.com` (flags) and
  `https://storage.ko-fi.com` (donation badge) — both self-hosted now.
- `link rel="preconnect"` / `dns-prefetch` for the above CDNs (no longer needed).

## [1.5.0] - 2026-05-11

### Added
- **Expand table** button in toolbar — opens the results table in a full-screen modal overlay with backdrop and Escape key support (8 languages: `expandTable` / `collapseTable`)
- Modal header bar with title and close (×) button, visible only when table is expanded
- `scripts/build.js` — simple Node.js file copy to `.cf-dist/` (no minification) for Cloudflare deployment
- `npm run build` and `npm run deploy` scripts in `package.json`
- `.cf-dist/` added to `.gitignore`

### Changed
- **"Friendly Name"** column renamed to **"Display Name"** across all 8 languages (`thFriendlyName`)
- Executive Summary in PDF redesigned: clean card with left blue border, improved typography, glossary section removed
- Cloudflare deployment now uses `npm run build` as the build command

### Removed
- **Markdown export** button and handler (MD export removed from UI and `app.js`)
- `execGlossaryMovable/Partial/NotMovable/Unknown` i18n keys (no longer used)

## [1.4.0] - 2026-05-01

### Added
- **Executive Summary** section in CSV and Markdown exports — matches the PDF report: total counts, percentages, recommendation text, and glossary

## [1.2.0] - 2026-04-27

### Added
- `SECURITY.md`: vulnerability disclosure policy
- `.gitattributes`: consistent EOL handling across platforms
- `scripts/update-database.ps1`: parses official Microsoft markdown docs and refreshes `MOVE_DB_RAW`
- `.github/workflows/update-database.yml`: weekly auto-update (cron) opening a PR when data changes
- Project structure: `LICENSE` (MIT), `.editorconfig`, `package.json`, `CONTRIBUTING.md`, `CHANGELOG.md`
- `scripts/` folder for automation utilities
- `tests/` folder with Vitest test suite (unit + integrity tests)
- GitHub Actions workflow for CI testing
- Database integrity validator (PowerShell, no Node required)

### Changed
- `scripts/minify.ps1`: now writes to `dist/` (gitignored) instead of next to source files
- Database source switched from third-party `tfitzmac/resource-capabilities` to **official `MicrosoftDocs/azure-docs`**
- Database refresh strategy: was live `fetch()` on page load, now offline auto-update via GitHub Actions
- Moved `minify.ps1` from root to `scripts/`

### Removed
- Tracked `*.min.css` / `*.min.js` files (now gitignored, regenerated on demand)
- Live `fetch()` IIFE in `js/move-database.js` (`MOVE_DB_LIVE_URL`, `moveDbSource`, `refreshMoveDB`)
- Third-party dependency on `tfitzmac/resource-capabilities`

## [1.1.0] - 2026-04-27

### Added
- Minified asset variants (`*.min.css`, `*.min.js`) — CSS -38%, JS -32%

- Schema.org `BreadcrumbList` structured data for SEO

- Improved Largest Contentful Paint via fetchpriority hint
- Reduced layout shift via explicit width/height on images
